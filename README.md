# Learn Java

A collection of basic Java programs for learning and practice.

## Files Included

- `ArraySum.java`: Calculates the sum of elements in an array.
- `Division.java`: Demonstrates basic division operations.
- `HelloWorld.java`: The classic introductory program.
- `MaxNumbArray.java`: Finds the maximum number in an array.
- `Printing.java`: Examples of different printing techniques.
- `productinputs.java`: Handles product inputs and calculations.
- `sum.java`: Basic addition of two numbers.

## How to Run

To compile any of these files, use:
```bash
javac FileName.java
```

To run the compiled class:
```bash
java FileName
```
